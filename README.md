# cv-generator
Web application (PHP/MySQL) for creating and managing professional resumes.

Para instalar esta aplicación sólo necesitas un servidor, una base de datos SQL y PHP instalado. XAMPP ofrece todo lo anterior.
Instalando XAMPP solo tienes que irte a la carpeta "data-base", 
copiar el contenido completo del archivo "schema-db.sql" y ejecutarlo en la terminal de XAMPP. 
O también puedes entrar en http://localhost/phpmyadmin/ pulsar la pestaña SQL, 
pegar el código SQL en la consola que ofrece la página y pulsar "Continuar". 

Con estos pasos crearás la base de datos para poder ejecutar la app web.
